# 📘 Concurso Guide — NotebookLM para Concurseiros

> Um guia inteligente desenvolvido com **NotebookLM** para atuar como um **segundo cérebro** na preparação para concursos públicos.

---

## 🚀 Acesse o Guia

👉 **NotebookLM (público):**  
🔗 https://notebooklm.google.com/notebook/be4a81d5-8eac-4767-9fdc-18c473786cd5

---

## 🧠 Objetivo do Projeto

- Ajudar concurseiros a **escolher o concurso ideal**
- Orientar na **organização e planejamento dos estudos**
- Centralizar informações relevantes em um único ambiente
- Reduzir tempo perdido com conteúdos dispersos

---

## 📚 O que você vai encontrar

- Curadoria de **vídeos e conteúdos estratégicos**
- Estrutura para **tomada de decisão (qual concurso escolher)**
- Direcionamento para **planejamento de estudos**
- Organização baseada em fontes confiáveis
- Apoio para revisão e compreensão dos temas

---

## 🎯 Público-alvo

- Iniciantes que não sabem por onde começar  
- Concurseiros com dificuldade de organização  
- Estudantes que buscam **eficiência e clareza**

---

## ⚙️ Como foi construído

- Uso do **NotebookLM** como base de organização e IA  
- Curadoria manual de conteúdos relevantes  
- Integração de múltiplas fontes (vídeos, sites e materiais)  
- Estrutura orientada para aprendizado ativo  

---

## 💡 Diferencial

Este projeto não é apenas um compilado de links.

É um **sistema de apoio à decisão e estudo**, projetado para reduzir esforço cognitivo e acelerar a jornada até a aprovação.

---

## 📌 Observação

O conteúdo foi construído com base em fontes públicas e confiáveis.  
Este material serve como apoio estratégico e não substitui estudo aprofundado.
